# AV Tech Solutions - Premium Audio Website

## 📋 Project Overview

A complete, modern, responsive multi-page website for an audio technology company. Built with **Tailwind CSS**, vanilla **JavaScript**, and **semantic HTML5**. No external dependencies except Tailwind CDN.

## 🚀 Customization

### To Add Your Content:
1. Replace placeholder text with your content
2. Update brand names and product info
3. Add your actual product images
4. Update contact information
5. Customize colors by modifying CSS variables

### To Modify Brands/Products:
Edit the `menuData` object in JavaScript:
```javascript
const menuData = {
    brands: [
        { id: 'brand-id', name: 'Brand Name', categories: [...] },
        // Add more brands
    ],
    products: {
        'brand-id': {
            'Category': ['Product 1', 'Product 2'],
            // Add categories and products
        }
    }
};
```
